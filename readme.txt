This is a tracker for Corona Virus data
Every hour it scrapes the Website of Ministry of Health of Govt of India and send a notification on the windows machine of the:
1. Cases of Indian Nationals
2. Cases of Foreign Nationals
3. Cured patients
4. Deaths
of the states specified by the user

Instructions:
Install following python packages:
1. Plyer
2. bs4
3. requests
4. time

On line 28, edit the states you want to recieve data for instead of the ones mentioned.
Run the program on your windows machine.
